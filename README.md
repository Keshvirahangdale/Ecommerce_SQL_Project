# Ecommerce_SQL_Project
Developed an E-Commerce Sales Analytics system using MySQL. Designed a normalized database with 6 relational tables and performed business analysis using joins, subqueries, CTEs, views, and window functions to generate sales and customer insights.


# E-Commerce Sales Analytics using SQL

## Project Overview

This project demonstrates the design and analysis of an E-Commerce database using MySQL. The database was created using normalized tables and analyzed using various SQL concepts to generate meaningful business insights.

## Database Tables

- Customers
- Categories
- Products
- Orders
- Order_Items
- Payments

## SQL Concepts Used

- Primary Keys and Foreign Keys
- Joins
- Aggregate Functions
- GROUP BY
- ORDER BY
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions (RANK)
- Views

## Business Insights Generated

- Total Revenue Analysis
- Top Selling Products
- Revenue by Category
- Customer Spending Analysis
- Customer Ranking
- Order Status Analysis
